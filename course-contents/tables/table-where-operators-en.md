| Operators (Predicates)  | Definition |
|---------------------------|-------------------------------------------------------------------|
| = | Equal |
| <> != | Not Equals |
| IN                        | are values in the subquery or in the given list?               |
| BETWEEN                   | Between given 2 values, limits are included|
| LIKE                      | Is given pattern in the string (wildcard are included) |
| AND                       | logical AND |
| OR                        | logical OR  |
| NOT                       | logical NOT |
